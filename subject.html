osu!

osu! is a very popular rhythm games that incorporates both rhythm and mouse movement. It is possibly one of the most popular rhythm games of all time, including having a very active esports scene.

StepMania

Initially developed as a clone of the popular arcade game Dance Dance Revolution, StepMania has since been developed into an open-source that is used by other games as its game engine.

Etterna

Etterna is very similar to StepMania, being programmed as an alternative to it. Etterna features more customization than StepMania, and not only is it able to import files from StepMania, it can also import files from osu!mania, a gamemode built into osu!
